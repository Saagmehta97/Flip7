# Flip7

A score tracker and leaderboard application for the game Flip 7.

## Description

Flip7 is a score tracking system that allows players to record and compare their game scores. The application features:

- **Score Tracking**: Add and record scores for all players
- **Leaderboard**: View all players ranked by their highest scores
- **Interactive Controls**: Use buttons to easily add new scores to the tracker

Players can input their scores through the interface, and the leaderboard will automatically update to show rankings from highest to lowest score.
